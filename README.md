# ✂️ Лендинг Барбершопа

Современный и стильный одностраничный веб-сайт для барбершопа, разработанный с акцентом на удобство онлайн-записи (в течение 30 секунд!) и высокую конверсию.

## ✨ Основной функционал

- 💈 **Адаптивный дизайн**: Полная поддержка мобильных и десктопных устройств.
- 📅 **Онлайн-запись**: Интерактивная система бронирования с использованием `flatpickr`.
- 💅 **UI/UX**: Продуманный пользовательский интерфейс с секциями:
  - Главный экран с призывом к действию.
  - Список услуг.
  - Блок «О нас» и гарантии.
  - Контактная информация.

## 🚀 Предпросмотр

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/m1sstak3/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
   ```
2. Откройте файл `index.html` в браузере или запустите через Live Server в вашей IDE.

## 🛠 Технологии

- HTML5
- CSS3 (Vanilla / Custom Properties)
- JavaScript (ES6+)
- `flatpickr` (для выбора даты и времени)

## 📝 Разработка

- `index.html` — основная разметка страницы.
- `style.css` — стили проекта.
- `script.js` — логика работы модальных окон, календаря и формы записи.
- `Project.md` — детальное описание требований и PRD (только для локальной документации).
