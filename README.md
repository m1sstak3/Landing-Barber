## ✨ Основной функционал

Лендинг спроектирован с учетом высоких стандартов UX/UI для локального бизнеса.

* 💈 **Адаптивный дизайн:** Полная и бесшовная поддержка мобильных, планшетных и десктопных устройств.
* 📅 **Быстрая онлайн-запись:** Интерактивная и интуитивно понятная система бронирования времени с использованием библиотеки `flatpickr`.
* 💅 **Продуманный UX:** Логичная структура секций, ведущая пользователя к целевому действию:
  * Главный экран с мощным призывом к действию (CTA).
  * Удобный и наглядный прайс-лист (Список услуг).
  * Блок «О нас» и гарантии качества.
  * Контактная информация и карта.

## 🛠 Технологии и архитектура

Проект написан на нативном стеке без использования тяжелых фреймворков для обеспечения максимальной скорости загрузки.

* **HTML5:** Семантическая разметка страницы.
* **CSS3:** Использование Custom Properties (переменных) для легкого управления темами, Flexbox/Grid для адаптивности.
* **JavaScript (ES6+):** Логика работы модальных окон, валидация формы и настройка календаря.

## 🚀 Предпросмотр и запуск

Проект не требует сборки. Для запуска достаточно выполнить пару шагов:

1. Клонируйте репозиторий:
```bash
git clone https://github.com/m1sstak3/barbershop-landing.git
```
2. Откройте файл `index.html` в вашем браузере.

*(💡 Для комфортной разработки рекомендуется запустить проект через расширение **Live Server** в VS Code).*
